## Why SUPER Skills For AI Vibe Coding

Super Data Analytics gives AI coding agents the context to build reliable data workflows, not just charts. Use it for pipelines, warehouse modelling, SQL tuning, dashboard design, KPI definitions, data quality, and measurement systems.

These skills are designed for **AI vibe coding**: fast, agent-assisted building where the AI needs strong domain context, practical workflows, guardrails, and implementation-ready outputs.

**SEO and discovery keywords:** data analytics, data engineering, BI dashboards, SQL optimisation, data pipelines, KPI dashboards, data quality, analytics engineering, AI vibe coding, agent skills.

## Install

Copy this folder into your agent's skills directory, then restart or reload the agent.

```bash
cp -R super-data-analytics ~/.your-agent/skills/
```

Use it by name:

```text
Use $super-data-analytics to help with this request.
```
